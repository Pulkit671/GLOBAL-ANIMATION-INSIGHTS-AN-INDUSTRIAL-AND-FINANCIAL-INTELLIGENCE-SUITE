## 📝 Project Overview
This interactive business intelligence suite delivers an end-to-end macroeconomic analysis of the global animation industry. By unifying decades of cinematic metadata with core financial metrics, the suite contrasts corporate capitalization models against long-tail consumer engagement trends. 

Utilizing dynamic dashboard action filters across geographic, temporal, and tier-based dimensions, the ecosystem shifts seamlessly from macro-level industrial overviews down to granular movie-level ledger entries. It serves as a tool for identifying high-ROI production models, historical medium adoption curves, and emerging international co-production hotspots.

🛠️ Technical Architecture & Tableau Feature Stack
Executive Performance Benchmarks (KPI Strip): Aggregates multi-billion dollar tracking metrics for Total Global Box Office and Total Invested Capital while dynamically counting unique studios, creative styles, producing nations, and languages.

Temporal Trend Analysis (Area Chart): Created continuous stacked area charts highlighting historical industry inflections—most notably the post-1995 transition from 2D traditional hand-drawn art to 3D computer-generated imagery (CGI).

Geographic Density Mapping (Density Heatmap): Implemented a multi-color neon temperature map layer utilizing custom washout techniques to expose global production cluster hotspots.

Relational Multi-Panel Matrixing (Bar Charts): Developed multi-measure row groupings to track the direct correlation between audience popularity scores, TMDB ratings, data quality metrics, and financial outcomes.

Dynamic Action Filtering (Interactivity): Implemented cross-sheet dashboard actions allowing interactive drill-downs where selecting any visual coordinate across charts automatically filters the entire data ecosystem in real time.
📂 Dataset Information
The visualization suite processes a comprehensive global animation film dataset, analyzing key variables including:

Financials: Budget, Box Office Revenue (USD Millions)

Creative Metadata: Animation Style (2D, 3D CGI, Stop-Motion), Studio, Original Language, MPAA Rating

Engagement Metrics: TMDB Popularity, TMDB Rating, Popularity Tier
